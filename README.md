# 🟢 BISLERI 3D BOTTLE MODEL

> A realistic **3D model of Bisleri water bottles**, created for design practice, visualization, and product modeling.

---

## 📌 Project Overview

This project showcases a **3D model of commonly used Bisleri bottles**, focusing on accurate structure, proportions, and visual detailing.
It is designed to help understand real-world product modeling and can be used for **rendering, animation, and visualization purposes**.

---

## 🎯 Objectives

* Build realistic **3D bottle models**
* Understand **product dimensions and structure**
* Practice **3D modeling and rendering techniques**
* Create assets for **animation or visualization projects**

---

## 🧃 Bottle Variants Included

* 250 ml (Compact Bottle)
* 500 ml (Standard Bottle)
* 1 Litre Bottle
* 2 Litre Bottle

---

## 🛠️ Tools & Technologies

* Blender / AutoCAD / Fusion 360 / any 3D modeling tool
* Basic 3D modeling techniques
* Rendering and lighting setup

---

## 📐 Key Features

* Realistic **3D structure and proportions**
* Smooth surface modeling and clean topology
* Basic material and texture application
* Ready for **rendering and animation**
